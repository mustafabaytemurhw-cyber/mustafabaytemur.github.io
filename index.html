Isolated High-Voltage Relay Driver & Monitoring Module

High-Voltage Isolated 16-Channel Relay Driver Board (0–60V input)

  • This project involved the design and development of a 16-channel industrial relay control board capable of operating from a wide 0–60V DC input range. The system was designed with industrial-grade protection, isolated communication, and EMI-conscious architecture to ensure reliable operation in electrically noisy environments. The board integrates power management, relay control, feedback monitoring, and CAN communication into a single platform suitable for industrial automation applications.
  • System Architecture Design: Designed the complete hardware architecture for a 16-channel industrial relay control system.
  • Wide Input Power Design: Developed a power architecture supporting 0–60V DC input operation with protection against transients and electrical disturbances.
  • Input Protection & EMI Filtering: Implemented TVS protection, common-mode choke filtering, and capacitor networks to improve immunity against conducted noise and surge events.
  • Multi-Stage Power Conversion: Designed a two-stage power supply architecture consisting of a high-voltage buck converter generating 12V and a secondary regulator generating a low-noise 3.3V rail for control electronics.
  • Microcontroller Integration: Selected and integrated an MCU with sufficient I/O resources and CAN communication capability for industrial control applications.
  • Isolated CAN Communication: Implemented an isolated CAN transceiver to improve communication reliability and protect the control system from bus-related noise and ground potential differences.
  • High-Side Switching Architecture: Developed an optically isolated relay drive structure utilizing NPN transistor stages and P-channel high-side switching devices.
  • Relay Feedback Monitoring: Designed feedback circuits that provide clean 3.3V logic-level status signals to the MCU using NPN and PNP transistor stages.
  • Signal Integrity Optimization: Optimized return current paths, grounding strategy, and signal routing to improve system stability and reduce susceptibility to interference.
  • PCB Layout Design: Designed the complete PCB layout in Altium Designer with emphasis on high-current routing, thermal management, EMC compliance, and manufacturability.
  • Protection & Reliability Engineering: Generated Gerber, drill, assembly, and fabrication files required for PCB production.
  • Prototype Validation: Performed design verification and hardware validation to ensure proper operation of power, communication, feedback, and relay control subsystems.
  • Project: Embedded Hardware Project

Reverse Engineering Projects

Motor Winding Control Board Reverse Engineering

  • This project started with a non-functional motor winding control board provided by a customer. Since no schematic, design files, or technical documentation were available, the board was reverse engineered to identify the root cause of the failure and recreate a fully manufacturable design.
  • Signal Tracing & Circuit Analysis: Achieved high detection stability under sunlight and ambient IR noise through 38 kHz–modulated 940 nm IR LED driving.
  • Schematic Reconstruction: The TSOP34838 receiver's band-pass filter and AGC architecture ensured that only the correct carrier frequency was processed, significantly reducing false triggers.
  • Altium Library Development: Upon detection, an electrically isolated, noise-free trigger output was delivered to the door controller via an AQY282SX PhotoMOS SSR.
  • PCB Redesign: Separate logic and trigger power domains were designed to protect control electronics from motor-induced EMI, voltage spikes, and transient currents.
  • Manufacturing Documentation: A full bridge rectifier on the power line provides polarity protection and enables operation up to 35V DC and 25V AC.
  • Production Coordination: Red LEDs provide standby indication, while blue LEDs offer clear user feedback during activation and triggering.
  • Assembly Verification: The complete sensing unit, power architecture, SSR drive stage, and LED indication system were fully designed by me, resulting in a compact and product-ready smart sensor module.
  • Project Delivery: Delivered the manufactured and assembled boards to the customer as a fully functional replacement solution.
  • Project: Embedded Hardware Project
  • Category: Reverse Engineering
